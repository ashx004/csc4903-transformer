# CSC4903 Transformer (Summer '26)

This repository is for the completion of **CSC4903 at Louisiana Tech University** during the **Summer '26 quarter**.

The project is a **Python recreation of the overall Transformer architecture** as introduced in Google's paper, **"Attention Is All You Need."**

## Reference Paper

- Original paper: https://proceedings.neurips.cc/paper_files/paper/2017/file/3f5ee243547dee91fbd053c1c4a845aa-Paper.pdf
